📌 Cách thêm mã này vào theme

  1. Thêm snippet vào layout theme
    Mở file: sections/header.liquid hoặc layout/theme.liquid
    Add the following line where global scripts or custom snippets are usually placed:
    {% render 'flintverse-customize' %}
  2. Đối với các store đã được tùy chỉnh
    Snippet flintverse-customize.liquid đã có sẵn trong theme. Bạn có thể thêm mã tùy chỉnh (JS, CSS) trực tiếp vào snippet đó.
  3. Để tắt / gỡ bỏ tính năng này
    Chỉ cần xóa dòng sau khỏi sections/header.liquid hoặc layout/theme.liquid:
    {% render 'flintverse-customize' %}
    Không cần thay đổi gì thêm.
