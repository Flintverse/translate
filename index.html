📌 How to add this code to your theme

  1. Include snippet in theme layout
    Open the file: sections/header.liquid or layout/theme.liquid
    Add the following line where global scripts or custom snippets are usually placed:
    {% render 'flintverse-customize' %}
  2. For stores we already customized
    The snippet flintverse-customize.liquid already exists in your theme. You can add more custom code (JS, CSS) directly into that snippet.
  3. To disable / remove this custom feature
    Simply delete the following line from sections/header.liquid or layout/theme.liquid:
    {% render 'flintverse-customize' %}
    No other changes are required.
