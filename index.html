📁 Tạo snippet và dán mã

1. Vào Snippets → Add a new snippet (tạo file mới).
2. Đặt tên file: flintverse-customize.liquid
3. Dán toàn bộ mã đã tạo ở trên vào file này và lưu lại.

📌 Cách thêm snippet vào theme

  1. Mở file layout
    Đi đến sections/header.liquid hoặc layout/theme.liquid.
  2. Thêm dòng include
    Add the following line where global scripts or custom snippets are usually placed:
    {% render 'flintverse-customize' %}
  3. Đối với store đã được tùy chỉnh sẵn
    Snippet flintverse-customize.liquid đã tồn tại. Bạn chỉ cần mở file đó và dán thêm mã mới vào.
  4. Để tắt / gỡ bỏ
    Xóa dòng {% render 'flintverse-customize' %} khỏi layout.
